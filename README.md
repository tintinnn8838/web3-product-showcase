# Web3 Product Showcase

Một dự án landing page giới thiệu sản phẩm Web3 theo phong cách hiện đại và dễ trưng bày trên GitHub.

## Chức năng hiện có

- Phần mở đầu nổi bật với thông điệp rõ ràng
- Khu giới thiệu các hướng sản phẩm như DeFi, NFT và portfolio dashboard
- Khối số liệu mô phỏng để tăng cảm giác sản phẩm thật
- Phần điểm mạnh và định hướng phát triển
- Lộ trình phát triển trình bày theo từng giai đoạn
- Khu kêu gọi hành động ở cuối trang

## Công nghệ sử dụng

- HTML
- CSS
- JavaScript

## Cấu trúc thư mục

```text
web3-product-showcase/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Cách chạy

1. Tải mã nguồn về máy
2. Mở file `index.html` bằng trình duyệt

## Hướng phát triển thêm

- Kết nối link trực tiếp tới các repo nổi bật
- Thêm hiệu ứng chuyển động mượt hơn
- Bổ sung chế độ sáng và tối
- Thêm phần giới thiệu cá nhân và công nghệ sử dụng
- Tích hợp dữ liệu thật từ các sản phẩm Web3 đang xây dựng
